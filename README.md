# How Install
Install the ArchLinux (or do it with your own hands) with:

```sh
arch-install.sh [/dev/sdx]
```
> IMPORTANT: This script only works on GPT systems

Install all dependencies and the dotfiles with: 

```sh
install-gaps.sh
```
